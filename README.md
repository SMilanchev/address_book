# Usage of the app:
## 1. Add 'AB_HOST' and 'AB_PORT' to the .env file:
### For example:
AB_HOST=0.0.0.0

AB_PORT=8000

## 2. Add interpreter of the project, recommend python 3.10+
## 3. Install requirements for the project:
### In terminal type: 
### ```pip install -r requirements.txt```
## 4. Start the project
### In terminal type:
### ```python api.py```
